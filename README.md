# Paintable.js

For Usage see the [docs](docs/README.md).

## development

Clone the repo and execute

```bash
yarn
yarn dev:all

# and one of the following
yarn example:vue
yarn example:react
yarn example:angular
yarn example:core
```

## build

```bash
yarn
yarn build:all
# yarn build:ci also builds the readme files in the dist directory
```
